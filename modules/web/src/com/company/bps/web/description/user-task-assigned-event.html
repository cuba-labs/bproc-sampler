The NotificationsSender bean listens for the UserTaskAssignedEvent spring event of the process with the user-task-assigned-event key. When the user task is assigned (the UserTaskAssignedEvent is fired) an email is sent.

How to use the sample

Click the Start process button. In the start process form, select a participant. There must be a user with a non-empty email (e.g. alice). After the process is started, an email will be sent. You may find the email in the Administration - Email history screen.
