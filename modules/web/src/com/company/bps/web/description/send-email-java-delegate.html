The process has a service task that is backed by the SendEmailJavaDelegate.

The SendEmailJavaDelegate defines three fields:

  • addressee
  • emailSubject
  • emailBody

Their values are set in the modeler and then used in the JavaDelegate source code.

How to use the sample

Click the Start process button. After the process is started, the process will reach the service task and send an email. You may find the email in the Administration - Email history screen.
