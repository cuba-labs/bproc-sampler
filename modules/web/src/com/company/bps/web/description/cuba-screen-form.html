The process demonstrates how to use CUBA screens as task and start process forms.

The SimpleStartForm.java demonstrates the following:

  • How to link a UI component with the process variable using the @ProcessVariable annotation
  • How to start the process, saving the values of the @ProcessVariable UI component as process variables

The SimpleTaskForm.java demonstrates the following:

  • How to use process form parameters (@ProcessFormParam)
  • How to complete a task with different outcomes
  • How to pass a process variable on task completion
  • How to define information for the modeler:
    • Which outcomes are used in the form
    • Which process variables may be set by particular outcome
    • Which process variables may be set by any outcome

How to use the sample

Click the Start process button. The CUBA screen start process form will appear. To see the CUBA screen task process form, open the BProc - My Tasks screen, find the task in the list and double-click the task.
